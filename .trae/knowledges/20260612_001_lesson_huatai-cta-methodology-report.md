# 华泰证券时序CTA方法论综述核心框架

**日期**: 2026-06-12
**来源**: 华泰证券金融工程专题，徐特、何康，2026-06-10
**规则文件**: .trae/rules/03-strategies/29-cta-architecture.md

## 核心思想

趋势可交易性的核心度量是趋势信噪比（TSI）及其 Bootstrap 置信区间。
CTA 策略必须遵循三层信号体系、四层退出机制与市场状态驱动。

## 关键结论

1. 收益率三要素分解：r_t = μ + ρ·r_{t-1} + ε_t，波动兼具外生噪声和内生混沌
2. 所有线性趋势指标=历史收益率的加权卷积：trend_t = Σc_s·δ_{t-s}
3. 三层信号：线性滤波层（带量纲）→ 状态变换层（去量纲）→ 复合信号层（多周期共振）
4. 四层退出：技术性止损 → 逻辑性证伪 → 时间性强制退出 → 系统性全局风控
5. 退出与市场状态联动：趋势→ATR移动止损，震荡→逻辑性止损
6. CTA是牛市策略，横截面是震荡/熊市策略，两者天然互补
7. 组合层面按卡玛比加权而非Sharpe

## 技术细节

**Bootstrap置信区间趋势判定**: TSI CI不跨零轴+ρ>0+σ<阈值=趋势，否则震荡

**Kaufman效率比**: ER=|close_t-close_{t-T}|/Σ|Δclose|, h=h_max-ER·(h_max-h_min)

**ATR移动止损**: TR做5%-95%截断后计算ATR，止损向浮盈方向移动永不回撤

**风险乘数M**: 沪深300ETF 4.5, 黄金ETF 3.0, 国债ETF 1.5

**震荡市证伪**: RSI>70开空→RSI回落至60平空，RSI<30开多→RSI回升至40平多

**评价指标**: 卡玛比=年化收益率/最大回撤（绝对收益导向，非打败基准）
